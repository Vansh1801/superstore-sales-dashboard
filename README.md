# Superstore Sales Performance Dashboard

## Overview
This project analyzes sales, profit, and return patterns using Power BI.

## Key Features
- Interactive dashboard with filters (Region, Category, Date)
- KPI tracking: Sales, Profit, Orders, Profit Margin, Return Rate
- Monthly profit trend to detect loss periods

## Key Insights
- Office Supplies has the highest number of returns
- Technology contributes highest sales
- Profit dips observed in certain months
- Furniture shows lower profitability

## Tools Used
- Power BI
- DAX
- Data Modeling
